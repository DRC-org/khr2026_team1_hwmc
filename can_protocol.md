# CAN 通信の仕様

* CAN ID は 11 bit まで（0x0～0x7FF；2048 種類）
* CAN は足回りとその他で 2 系統用意する（M3508 のフィードバックが通信を圧迫するため）
* 分かりやすさのため、系統をまたいでも ID は重複させない
